Main-Structure

B) CSS Files and Structure - top

This template includes a number of CSS files that are used for display the website. All CSS files, can be found in the Template's assets/css/ folder and some plugins folder.

The CSS file :

  1. bootstrap.min.css - It includes the based on Bootstrap Framework from Twitter ( Original code )
  2. themes.css - modify from based on Bootstrap Framework 3.02 for this theme,
  3. styles.css - This style sheet contains specific styling for the template structure. It is the base styles color , text styles, header and include style sheet some plugin
